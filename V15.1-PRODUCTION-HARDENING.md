# JobNova V15.1 production-hardening pass

This pass focuses on the highest-value stability improvements without changing the overall product structure.

## Included in V15.1
- stronger job-posting platform detection and browser fallback for generic sites
- improved generic extraction selectors for more public ATS/career pages
- better binary cleanup for weak PDF/DOC fallback text recovery
- richer ATS scoring with evidence-density and score breakdown metadata
- role-family interview question banks and stronger final practice plans
- safer default resume layout for Job Ready: one-page ATS-first
- version bump to 15.1.0

## What this does not claim
- full end-to-end runtime validation of every native mobile dependency
- live verification of third-party integrations such as OpenAI, storage, email, or OCR credentials
- guaranteed perfect extraction from every protected job board

## Highest-value next step after V15.1
Focus on resume extraction truthfulness, structured ATS comparison history, and a tighter route cleanup pass so the app surface matches the strongest core flow: Career Path -> Resume Match Lab -> Tracker -> Interview Coach.
