
# V7.4 Hardening Notes

This pack adds:
- request IDs and structured telemetry
- Zod schema validation for ATS, export, and realtime interview routes
- in-memory queue stubs for export and interview scoring jobs
- Postgres-ready adapter with local JSON fallback
- stronger error handling and safer health diagnostics

## Environment additions
- DATABASE_URL=
- PGSSL=true
- TELEMETRY_ENABLED=true

## Production next steps
1. Replace local JSON store writes with Postgres queries.
2. Move queue stubs to BullMQ or another durable queue.
3. Send telemetry to Sentry, OpenTelemetry, or another collector.
4. Add retry policies for OpenAI, Azure, and Supabase operations.
